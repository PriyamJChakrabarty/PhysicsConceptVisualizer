# Physics Sphere: Physics Concept Visualizer

An interactive browser-based simulation tool for learning classical physics using 3D animations. Built using Three.js and WebGL.

## 📌 Overview

**Physics Sphere** helps students and educators explore core physics concepts through real-time, interactive 3D simulations such as projectile motion, pendulum oscillation, buoyancy, and more.

## 🌍 Live Deployment

Access the visualizer online here:  
🔗 [https://priyamjchakrabarty.github.io/PhysicsConceptVisualizer/](https://priyamjchakrabarty.github.io/PhysicsConceptVisualizer/)

## 🚀 Features

- **Six Physics Simulations**
  - Projectile Motion
  - Relative Motion
  - Rotational Motion
  - Simple Pendulum Motion
  - Spring Motion
  - Buoyancy Simulation 

- **User Interactions**
  - Input sliders and fields for parameters like angle, velocity, mass, etc.
  - Start, Pause, Reset buttons
  - 3D camera: orbit, zoom, pan
- Smooth animation (≥ 30 FPS)
- Fully client-side: no backend required

## 🧰 Tech Stack

- JavaScript, HTML5, CSS3
- [Three.js](https://threejs.org/)
- WebGL

## 📥 Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/PriyamJChakrabarty/PhysicsConceptVisualizer.git
   cd PhysicsConceptVisualizer
   ```
2. Open `index.html` in a modern web browser.

## 🌐 Compatibility

- Works on Chrome, Firefox, Edge (latest versions)
- No internet required after first load
- Requires WebGL support
